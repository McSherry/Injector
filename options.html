Injector

Version 0.4.0

Injector is a helpful utility allowing users to add custom CSS rules into websites of their choice.

It was created after this functionality (achieved via Custom.css) was removed from Chrome in versions 32 onward. Injector provides much the same functionality, although pages will have to be reloaded when rules are modified, as they do not update "on-the-fly" as Custom.css rules did.
